```
  01  ·  Architect first. Code second. Refactor always.
  02  ·  Non-blocking, async backend architecture — by default, not by accident.
  03  ·  Database schemas are contracts. Design them with intent.
  04  ·  Optimize queries and indexes from day one — not retrofitted later.
  05  ·  Clean, modular, self-documenting code over clever shortcuts.
  06  ·  Performance is a feature. Treat it as one at every layer of the stack.
```
